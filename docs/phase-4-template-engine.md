# Phase 4: Template Engine

Phase 4 creates the first real scaffold path from kernel catalog to generated project skeleton.

## Built Artifacts

- `template-manifest.schema.json`
- `template-list.mjs`
- `template-scaffold.mjs`

## Current Capability

The scaffold engine can:

- list all catalog templates
- validate template to QA profile mapping
- create a project skeleton under `generated/`
- write `.sage/project-plan.json`
- write starter docs and environment placeholder
